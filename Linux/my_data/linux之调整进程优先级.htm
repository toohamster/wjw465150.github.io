ps -eo nice,pid,cmd #显示优先级值，进程id以及命令名
ex:[root@home root]# ps -eo cmd,pid,nice |grep gedit
gedit 2406 5
grep gedit 2410 0

nice -n N command #按优先级N(-20~19)执行程序
ex:[root@home root]# nice -n 5 gedit

renice [+N | -N ] process #调整正在运行的进程的优先级(N)
注意：+N代表减低优先级，-N代表提高优先级。
ex:[root@home root]# renice -2 2406
2406: old priority 5, new priority -2

[root@home root]# ps -eo cmd,pid,nice |grep gedit
gedit 2406 -2
grep gedit 2413 0

 
或者使用top直观地调整nice。
键入top命令，按r
