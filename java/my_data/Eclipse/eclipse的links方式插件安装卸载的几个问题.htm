eclipse的links方式插件安装卸载的几个问题

用links目录方式安装eclipse插件时需要注意的问题:
1.新增加的插件文件路径可以添加在links目录下其他文件中,也可以单独建立一个文件,形式都是path=...,只要重新启动eclipse就可以生效.但是需要注意的是如果按照前一种方式,当执行eclipse -clean时候,若一个文件放置多个path路径,则只有最后一个生效.
2.对于eclipse3.0,删除configuration目录重新启动eclipse与直接执行eclipse -clean的效果是一样的,但是对于eclipse3.1不可以直接删除configuration目录,必须保留config.ini文件.
3.eclipse -clean启动方式,一般是用于插件去除后的还原操作,因为仅仅去除links目录里的路径指示有时是不够的,可能会导致启动错误.还有一种用途,就是在安装多国语言包后,因为之前已经启动过英文版的eclipse,导致虽然插件安装,但是汉化不彻底.用eclipse -clean重新启动可以改变这种状况.
