jps为java查看java进程的命令行工具。支持远程查看，命令行格式为：

jps [ options ] [ hostid ]


options 包括如下参数：

-q：列出进程号

 

 

Java代码  
  1. [ ~]$ jps -q  
  2. 10818  
  3. 13406  
  4. 10031  
  5. 19284  

 

-m：列出main方法的参数

 

Java代码  
  1. [ ~]$ jps -m  
  2. 10818 TaskTracker  
  3. 13406 Server --a.Config=/af/ice_icebox.properties  
 

-l：jar和main方法的全路径

 

Java代码  
  1. [ ~]$ jps -l  
  2. 10818 org.apache.hadoop.mapred.TaskTracker  
  3. 13406 aBox.Server  
 

-v：传递给jvm的参数

-V：输出通过标记文件传给jvm的参数

-Joption：输出传递option参数给java启动器(由jps调用)。


hostid：

[protocol:][[//]hostname][:port][/servername]
